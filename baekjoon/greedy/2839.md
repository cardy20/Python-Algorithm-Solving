# 문제 
2839 설탕배달

# 시간
- 48분 - 재풀이

# 코드
```python
N=int(input())
F=N//5
N%=5
T=0
while F >=0:
    if N%3==0:
        T=N//3
        N=N%3
        break
    F=F-1
    N=N+5
print((N==0) and (F+T) or -1)
```
# 풀이

- Greedy 방식 풀이 
  - 일단 5로 모두 나누고 본 다음에, 3으로 나누어 값을 조정한다.
  - 이 때, 초기값은 5로 나눈 몫(F) 남은 값을 기준으로 3으로 나눠지지 않으면 F를 조정하고 이를 반복한다. 
  - 알고리즘이 모두 끝나고 N이 0이라면 최종 몫은 정답이 되고 이에 해당하는 조건이 아니면 -1을 출력한다.
